# Unsupervised Learning

This folder contains datasets, notebooks, and project work for Unsupervised Learning techniques, including:

- **K-Means Clustering**  
- **DBSCAN Clustering**  
- **PCA (Principal Component Analysis)**  
- **Basic EDA (Exploratory Data Analysis)**  

## Datasets

1. **Mall Customers**  
   - [Mall_Customers.csv](https://raw.githubusercontent.com/marcopeix/cluster-analysis/master/Mall_Customers.csv)  
   - Used for clustering customers based on Age, Annual Income, and Spending Score  

2. **Credit Card Customers**  
   - [Credit_Card_Customers.csv](https://raw.githubusercontent.com/plotly/datasets/master/cc_data.csv)  
   - Used for clustering customers based on credit card features
